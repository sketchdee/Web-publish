C#系统时间相关的类 C# 系统时间

获取日期+时间
string currenttime = DateTime.Now.ToString(); // ToString() 方法可以不用加
DateTime.Now.ToLocalTime()//另外的一种方法

获取日期
string curentdate = DateTime.Now.ToLongDateString();
DateTime.Now.ToShortDateString();
DateTime.Now.ToString("yyy-MM-dd")

获取时间
DateTime.Now.ToLongTimeString()
DateTime.Now.ToShortTimeString()
DateTime.Now.ToString("hh:mm:ss")
DateTime.Now.TimeOfDay.ToString()

其他时间获取
DateTime.ToFileTime().ToString()
DateTime.Now.ToFileTimeUtc().ToString()
DateTime.Now.ToOADate().ToString()
DateTime.Now.ToUniverdalTime().ToString()

单独获取
下方都为int类型
DateTime.Now.Year //获取年份
DateTime.Now.Month //获取月份
DateTime.Now.DayOfWeek //获取星期（Thursday）
DateTime.Now.DayOfYear //获取一年的第几天
DateTime.Now.Hour //获取小时
DateTime.Now.Minute //获取分钟
DateTime.Now.Second //获取秒数
